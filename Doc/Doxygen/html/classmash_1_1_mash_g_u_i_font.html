Creates a renderable version of the given text.

The result of this function can be used to render text to the screen. This method can be slow and is best used infrequently or with small strings.

This function assumes verticesOut has enough room to hold the final data. CalculateVertexBufferSizePrecise() or CalculateVertexBufferSizeQuick() can be called prior to calling this function to determine what size may be needed.
