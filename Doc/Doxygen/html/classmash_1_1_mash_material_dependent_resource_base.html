Detailed Description

This class is used internally by various objects that are dependent on particular API objects to be compiled. The root of this dependency is the material.

This helps to stall loading of API specific elemenets until all material and scene initialization is complete to eliminate unneeded compiling.

This class should never need to be accessed by the user.
