Detailed Description

This class deals with loading/saving files from different API's and API formats, and automatically handing loading from API or Virtual file system (VFS) directories.

The VFS can be used for any purpose including adding files from an archive using AddFileToVirtualFileSystem(). The files can then be read from a MashFileStream or ReadFile().
