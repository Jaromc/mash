Detailed Description

Can be used to create some basic meshes or manipulate current ones.

Member Enumeration Documentation

enum mash::MashMeshBuilder::eMESH_UPDATE_FLAGS
Enumerator
aMESH_UPDATE_NORMALS 

This will use the triangle layout of a mesh to calculate new normals. If the mesh is made up of shared positions using indices then surface normals will be smooth. However, if the mesh is made up of non-shared vertices then the resulting normals will not be smooth, each face will be defined when light hits it.

aMESH_UPDATE_FILL_MESH 

This flag will set mesh data even if no data needs to be updated. This is useful when creating new meshes. You can just pass in an empty mesh and let the update funtion fill it for you.

Updates a mesh based on the flags given.

This function can create a mesh from raw data.

It is safe to pass pointers into the sMesh struct that live in the destination mesh. Internally this function will duplicate the data only if it has detected that duplication is necessary.
