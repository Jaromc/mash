bool mash::sLoadSceneSettings::deleteMeshInitialiseDataOnLoad

Initialise mesh data is used for some mesh manipulation functions and material changes on initialization. Setting this to true will call MashMesh::DeleteInitialiseData() as soon as loading is done with it. This can be handy to reduce memory footprints with large scene while loading. If loading from .nss files then you probably have already set the scene up how you want it so it would be safe to set this to true.

Note if loading is being done within MashGameLoop::Initialise() then MashMesh::DeleteInitialiseData() will automatically be called when that function exits.

saveGeometryFlags Can be set to decide what data is deleted when MashMesh::DeleteInitialiseData() is called.

uint32 mash::sLoadSceneSettings::frameRate

Sets the animation frame rate of any transform animations. This is only valid when loading COLLADA files.

uint32 mash::sLoadSceneSettings::saveGeometryFlags

Sets what data will be saved within the mesh when MashMesh::DeleteInitialiseData() is called. These flags can be fund in eSAVE_MESH_DATA_FLAGS. This can also be set in MashMesh::SetSaveInitialiseDataFlags() after loading.
