Detailed Description

The controllers are resposible for interpolating there data to a particular frame. They are accessed by the animation mixer and users shouldn't really need to use them directly.

These hold animation key data for a node instance. So Each node will have a unique controller, but the keys held internally are shared to each controller.

Controllers are commonly used for node interpolation. But others can be created to handle animations such as light colours.

Once a key set has been assigned to a controller the key set should not be modified.
