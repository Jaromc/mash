Detailed Description

An XML reader can be created from MashFileManager::CreateXMLReader(). When your done with it call Destroy().

This reader assumes you know the general layout of the XML file.

Example usage:

<RootNode> <Hand handname="MyBigHand" handsize="10.0"> <Finger fingername="FirstFinger"> <Finger fingername="SecondFinger"> </Hand> </RootNode>

if (MoveToFirstChild("Hand")) { MashStringc handName; f32 handSize = 0.0f;

GetAttributeString("HandName", handName); GetAttributeFloat("HandSize", handSize);

if (MoveToFirstChild("Finger")) { do { MashStringc fingerName; GetAttributeString("FingerName", fingerName);

}while(MoveToNextSibling("Finger"));

PopChild(); }

PopChild(); }
