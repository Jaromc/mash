Reserves memory.

This can be used to make adding elements more memory efficient. This does not change the Size() of this array.

This will not reduce the number of reserved elements in this container. Use DeleteData() or ShinkToFit() intead.

void mash::MashArray< T, TAlloc >::Resize ( uint32  elementCount,
const T &  val = T() 
)
inline

Changes the Size() of this array.

This will not reduce the number of reserved elements in this container. Use DeleteData() or ShinkToFit() instead.

If the new number of elements is less than the current value then those extra elements will have their destructors called. Char types are optimised to use memset and set to 0.

Parameters
elementCountNew Size().
valNew elements will be initialised to this value.
